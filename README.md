# 项目目录
```text
|- project
|-- annotation 自定义注解
|-- model 基础类信息
|--- entity 实体
|--- enums 枚举
|--- pojo 通用类信息
|--- request 请求信息
|--- response 返回信息
|-- config 配置
|-- controller 服务接口
|--- api app
|--- manage 管理后台
|-- datasource 数据
|--- mapper mybatis
|---- filter 筛选参数
|---- typeHandler 转换器
|--- repository jpa
|-- service 服务业务逻辑
|--- imp 实现
```
# 项目基础
springboot 单体应用项目、数据库mysql、缓存redis
